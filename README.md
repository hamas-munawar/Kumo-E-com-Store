# Kumo E-commerce Store

A modern e-commerce website built with HTML, CSS (Tailwind), and JavaScript.

## Features

- Responsive design
- Product catalog
- Image gallery
- Modern UI components

## Project Structure

```
├── public/
│   ├── image/         # Image assets
│   ├── index.html     # Main HTML file
│   ├── output.css     # Compiled Tailwind CSS
│   └── script.js      # JavaScript functionality
├── src/
│   └── input.css      # Tailwind source CSS
└── tailwind.config.js # Tailwind configuration
```

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```

## Development

To compile Tailwind CSS:
```bash
npm run build:css
```

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Node.js (for development)
